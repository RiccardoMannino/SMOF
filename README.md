# San Martino Outdoor Festival

Questo sito è stato creato con Sanity e Next.js

## Getting Started

First, run the development server:

```bash
npm run dev
```

- Apri [http://localhost:3000](http://localhost:3000) con il tuo browser per vedere il risultato.
- Apri [http://localhost:3000/studio](http://localhost:3000/studio) per modificare i contenuti.
